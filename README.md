## 框架主要新特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 更强大和易用的查询
* 全新的事件系统
* 模型事件和数据库事件统一纳入事件系统
* 模板引擎分离出核心
* 内部功能中间件化
* SESSION/Cookie机制改进
* 对Swoole以及协程支持改进
* 对IDE更加友好
* 统一和精简大量用法

##  开发特性
* 多种API
* 独立后台
* 独立代理后台
* 代理独立项目或多项目
## 安装

~~~
将源码上传到服务器后进行解压访问按照步骤逐步安装
~~~
* 将运行目录设定为Public
* 配置根据环境进行配置伪静态
